DIY Syringe Pump Hardware Connection Overview

This page matches the current diy-syringe-pump.ino sketch for Arduino UNO R4 Minima. It assumes the 2.4/2.8 inch Uno-style parallel TFT shield is installed, and that the motor is driven through an Adafruit TMC2209 breakout with a single half-duplex UART pin.

Target board: UNO R4 Minima Display: Uno parallel TFT shield Driver: Adafruit TMC2209 breakout Current sketch uses UART readback

TMC2209 Breakout

  • UNO R4 D10 -> TMC2209 STEP
  • UNO R4 D1 / TX1 -> TMC2209 UART net
  • UNO R4 D0 / RX1 -> same TMC2209 UART net
  • UNO R4 5V -> TMC2209 VDD
  • UNO R4 GND -> TMC2209 GND
  • Motor PSU + -> TMC2209 terminal +
  • Motor PSU - -> TMC2209 terminal - / GND
  • Stepper coil A -> 1A / 1B
  • Stepper coil B -> 2A / 2B

Endstops and Buzzer

  • Forward endstop -> A5
  • Backward endstop -> D11
  • Buzzer + -> D13
  • Buzzer - -> GND

The TFT shield already occupies D2-D9 and A0-A4. Those pins are not free for other hardware.

Important TMC2209 UART note: this sketch does more than send commands; it also verifies readback from the driver. Because the Adafruit breakout has a single UART pin, that pin has to sit on a shared half-duplex UART net that reaches both D1/TX1 and D0/RX1. A one-way TX-only connection is not enough for this sketch.

What The Sketch Actually Uses

Used on the TMC2209

  • STEP is used directly from D10.
  • UART is used for configuration, enable/disable, and motor direction.
  • VDD, GND, and motor power are required.
  • 1A / 1B / 2A / 2B go to the motor coils.

Not used by this sketch

  • DIR is not wired because direction is sent over UART.
  • EN is not wired because enable is sent over UART.
  • DIAG and INDEX are not connected.
  • MS1 / MS2 are not required because microstepping is configured over UART.

If UART is not working, the sketch will usually stop with a TMC2209 UART error instead of starting the pump.

Endstop and Power Notes

Endstop logic: the sketch sets both endstop inputs to INPUT_PULLUP and treats a LOW reading as active. That lets the motor run even when no endstop is connected, and matches a normally open (NO) switch wired between the pin and GND.
  • If your endstops are normally closed (NC), the current logic will not match and you should either rewire them or invert the logic in the sketch.
  • The TMC2209 motor power should come from the motor supply, not from the UNO 5V pin.
  • Motor coil order is not defined by the sketch. If direction or motion is wrong, swap one coil pair or adjust the motor direction setting in software.
  • The sketch configures the TMC2209 for 32 microsteps over UART and expects address 0.
